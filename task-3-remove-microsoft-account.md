# Task 3: Remove Microsoft Account and Revert to Local Account

## Objective
Remove Microsoft account authentication and revert the device
back to a local account.

## Steps Performed
1. Signed in using Microsoft account
2. Navigated to:
   Settings → Accounts → Your info
3. Selected "Sign in with a local account instead"
4. Verified identity using Windows Hello PIN
5. Reconfigured local account credentials
6. Signed out and verified local sign-in

## Outcome
Microsoft account authentication was successfully removed,
and the device was restored to local account sign-in.
