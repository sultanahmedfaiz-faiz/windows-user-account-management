# Task 1: Configure and Manage Local User Accounts

## Objective
Create two local user accounts on a Windows 11 device:
- User1: Local Administrator
- User2: Standard User

## Steps Performed
1. Signed in as local Administrator
2. Navigated to:
   Settings → Accounts → Family & other users
3. Created User1 as a local account
4. Changed User1 account type to Administrator
5. Verified group membership via Computer Management
6. Created User2 as a standard local user
7. Signed in as User1 to create the user profile
8. Verified profile creation under `C:\Users`

## Verification
- User1 is a member of the Administrators group
- User2 exists as a standard user
- User profile folders successfully created

## Outcome
Local user accounts were successfully configured with appropriate
permissions following least-privilege principles.
