# Windows User Account Management (Windows 11)

## Project Overview
This project demonstrates hands-on experience managing local user accounts
and Microsoft account authentication on a Windows 11 device.

The lab simulates real-world IT Support and Endpoint Administrator tasks,
including local user creation, privilege assignment, Microsoft account
integration, Windows Hello PIN configuration, and account rollback.

## Environment
- Operating System: Windows 11
- Device Name: SEA-WS3
- Account Types: Local accounts, Microsoft account
- Authentication: Password and Windows Hello PIN

## Skills Demonstrated
- Local user and group management
- Administrator vs standard user configuration
- Microsoft account integration
- Windows Hello (PIN) configuration
- Account security and authentication management
- IT documentation and troubleshooting mindset

## Tasks Covered
- Task 1: Configure and manage local user accounts
- Task 2: Configure Microsoft account authentication
- Task 3: Remove Microsoft account and revert to local account
