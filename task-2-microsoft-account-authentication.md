# Task 2: Configure Microsoft Account Authentication

## Objective
Convert a local administrator account into a Microsoft account
and configure Windows Hello PIN authentication.

## Steps Performed
1. Signed in as User1 (local administrator)
2. Navigated to:
   Settings → Accounts → Your info
3. Switched from local account to Microsoft account
4. Signed in using Microsoft account credentials
5. Configured Windows Hello PIN authentication
6. Verified successful sign-in using PIN

## Authentication Method
- Microsoft Account
- Windows Hello PIN

## Outcome
The device was successfully linked to a Microsoft account,
enabling cloud-based authentication and enhanced security.
